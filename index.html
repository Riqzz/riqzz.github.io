使用Hexo搭建github博客

Posted on 2021-09-28

1. 准备

本文参考这篇文章，默认已具备一个github账号，安装了Node.js，npm，git

整个过程使用 git bash 环境。

注：Node.js 版本低于v12可能会在后面执行 hexo g 命令时报错，需要升级版本。

  • Win10 升级 Node.js 可直接在官网下载新版覆盖安装
  • 升级 npm 可使用命令 npm install -g npm

2. 创建一个仓库

仓库名称为 用户名.github.io，这也是将来使用的域名。

3. 使用Hexo

查看官方文档戳这里

3.1 博客模板

使用命令安装 Hexo

创建一个目录来保存源代码（包括hexo和博客的markdown源码）并初始化

hexo 会自动下载一些文件

而后使用命令生成相应的html文件

使用命令可以在本地 4000 端口启动这个服务进行预览

3.2 更改主题

原主题不好看，可以安装其他主题，以Next主题为例，使用如下命令将主题下载到themes中

修改 _config.yml 文件的theme: landscape 为 theme: next ，而后重新生成

4. 上传

修改 _config.yml 文件的 deploy:项配置git （这里默认ssh已经配好了的）

而后使用命令发布，而后就可以在仓库中看到对应的html代码了（并不是hexo的源码）

如果出现报错Deployer not found: git，则需要安装支持

注意这时候打开域名 用户名.github.io 可能看不到你的修改（跟本地预览不同），需要几分钟的时间才会更新

5. 写文章

使用命令来生成一个markdown文章模板（这篇文章也是在测试这一项）
